About the Server

Toasty Craft A fun Server With Few Rules Just Join And Have Fun

Griefing, ambushes, traps and base hunting are part of the experience. Protect your builds, make allies, or vanish into the wild.

  • Game mode: Survival
  • PVP: Enabled everywhere

Server Commands

Basic public commands available on Toasty Craft. (i think this is all of them)

  • /dupe – (REMOVED).
  • /sethome – Set your home location.
  • /home – Teliports you to a set location
  • /delhome – Removed home location
  • /tp-p help - Shows all /tp-p commands and their descriptions.
  • /tp-p - Sends a request to teleport to the specified player.
  • /tp-p here - Sends a request for a player to teleport to you.
  • /tp-p accept - Accepts the most recent teleport request sent to you.
  • /tp-p dissable - Rejects and disables all incoming teleport requests to you.

Some commands may change as the server updates. Check Discord for the most accurate list.
